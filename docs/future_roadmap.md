# SubsqManager 今後の改修案・ロードマップ (v1.1以降)

ユーザーフィードバックやペルソナ分析から得られた、今後のアップデートで追加すべき機能や改善案のストックです。

## 🎯 最優先UX改善 (v1.1予定)

### 1. 入力フォームの「プログレッシブ・ディクロージャー（段階的開示・蓋閉じ）」
- **目的**: ズボラなユーザーでも直感的に使えるよう、入力フォームの心理的ハードルを下げる。
- **実装内容**: 
  - 追加フォームのデフォルト表示を「サービス名」「金額」「周期」「カテゴリ」の必須4項目のみにする。
  - その下に「詳細設定を開く（任意）」トグルボタンを設置し、タップした時のみ「満足度」「利用頻度」「割り勘設定」「トライアル期間」「終了日」などを表示する。

## 💡 新機能・インサイト (v1.2以降)

### 2. 「これ使ってないのでは？」AIインサイト＆解約提案
- **目的**: 節約のモチベーションをダイレクトに刺激する。
- **実装内容**: 
  - ユーザーが設定した「利用頻度（月に1回未満など）」や、前回の使用からの経過日数を分析。
  - ダッシュボード上部にアラートとして「⚠️ Netflixの利用頻度が低いようです。今月は一時停止（削減）しませんか？」といった提案カードを表示する。

### 3. 節約モチベーションを高めるゲーミフィケーション
- **目的**: アプリを触る楽しさを増やす。
- **実装内容**:
  - サブスクを「削減」した瞬間に紙吹雪のアニメーションを実装。
  - ダッシュボードトップに「🎉 先月より固定費が 2,400円 浮きました！」などのポジティブなメッセージを表示。

## 💼 高度なカスタマイズ (v1.3以降)

### 4. 「経費」と「プライベート」の完全分離（タグ機能の進化）
- **目的**: フリーランスや個人事業主のニーズに対応する。
- **実装内容**:
  - 各サブスクリプションに「経費フラグ（トグル）」を追加。
  - ダッシュボードの集計で「経費を除外して計算する」フィルタリング機能を実装し、純粋な個人の生活費を正確に把握できるようにする。

### 5. 「家族シェア・割り勘」の実質負担額・完全可視化
- **目的**: 家族でのサブスク共有時の家計管理を正確にする。
- **実装内容**:
  - 現状の割り勘設定を活かし、ダッシュボードの表示金額を「サービス総額ベース」と「自分の実質負担額ベース」でワンタップで切り替えられるUIを追加。

### 6. 外貨（ドル建て等）対応と為替概算
- **目的**: 海外SaaSやツールのサブスク管理。
- **実装内容**:
  - 金額入力時に通貨（USD, EUR等）を選択できるようにし、リアルタイム為替（または固定レート）で日本円に換算してダッシュボードに合算する機能。
