# プライバシーポリシー (Privacy Policy)

「コテサク - サブスク管理・固定費削減」（以下、「本アプリ」といいます）は、ユーザーのプライバシー保護を最重要視しています。本アプリの機能をご利用いただくにあたり、ユーザーデータの取り扱い方針を以下に定めます。

## 1. 取得する情報および利用目的

本アプリは、利便性向上のために以下のデバイス機能にアクセスしますが、**データはすべてユーザーの端末内（オンデバイス）で処理され、外部サーバーへの送信・保存は一切行いません。**

*   **写真・カメラへのアクセス (スクリーンショット解析)**
    *   **利用目的**: レシートやスクリーンショットの画像から、サービス名や金額等のテキストをOCR（光学文字認識）で抽出し、サブスクリプション登録フォームに自動入力するため。
    *   **データの取り扱い**: 画像データおよび抽出されたテキストデータは、Appleが提供するデバイス内のVisionフレームワークを用いてローカルで処理されます。開発者や第三者のサーバーに送信されることはありません。
*   **カレンダーへのアクセス**
    *   **利用目的**: ユーザーが登録したサブスクリプションの次回請求日やトライアル終了日を、iOS標準カレンダーに予定として登録するため。
    *   **データの取り扱い**: 端末のカレンダーアプリに対して予定を追加する目的でのみ使用します。
*   **通知へのアクセス**
    *   **利用目的**: サブスクリプションの請求日やトライアル終了日が近づいた際に、リマインダー通知を送信するため。

## 2. ユーザーデータの保存と同期

*   **ローカル保存**: アプリに入力されたサブスクリプション情報等のデータは、端末内のローカルストレージ（SwiftData / Core Data）に保存されます。
*   **iCloud同期**: ユーザーがご自身のApple ID設定でiCloudバックアップおよび同期を有効にしている場合、データはユーザー個人のiCloudアカウントに暗号化されて保存・同期されます。開発者がこのデータにアクセスすることは不可能です。

## 3. 第三者への情報提供

本アプリは、ユーザーの個人情報や入力データを第三者に提供、販売、共有することはありません。
※ 本アプリ内での解析処理（OCR等）は外部のAIサーバーではなく、すべてiOSデバイス上のネイティブAPIを使用しています。
