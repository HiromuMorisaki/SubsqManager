# SubsqManager (コテサク)

「コテサク」は、プレミアムなUI/UXとスマートな分析機能で、固定費（サブスクリプション）を「楽しく、賢く」管理・削減するためのiOSアプリです。

## 🌟 主な機能

- **ダッシュボード＆コスト可視化**
  - 月額・年額の支払い合計をシームレスに切り替え表示。
  - カテゴリ別（エンタメ、音楽、仕事など）の支出割合を美しいドーナツグラフで分析。
- **爆速サブスク登録 & AIスクショ解析**
  - **爆速クイック追加**: Netflix, Spotifyなどの人気サービスは1タップでプランと金額を自動入力。
  - **AIスクショ解析 (OCR)**: 領収書や明細のスクリーンショットを選ぶだけで、AppleのVisionフレームワークがサービス名、金額、請求周期を自動抽出してフォームを埋めます。
- **削減履歴と1タップ復元 (一時停止機能)**
  - 使っていないサブスクを「削減」すると削減履歴タブに移動し、いくら節約できたかを可視化。
  - 再契約したくなった時は、履歴から右スワイプの「1タップ復元」で即座にアクティブに戻せます。
- **通知リマインダー＆カレンダー連携**
  - 請求日の数日前（設定で変更可能）にプッシュ通知でお知らせ。
  - iOS標準カレンダーに次回請求日を自動同期。無料トライアルの終了忘れを防止します。
- **プレミアムなUI体験**
  - iOS 17の機能をフル活用したグラスモーフィズムデザイン。
  - 成功時のHaptic Feedback（触覚フィードバック）や滑らかなアニメーション。

## 🛠 技術スタック

- **Target**: iOS 17.0+
- **Frameworks**: SwiftUI, SwiftData
- **Native APIs**: Vision (OCR), PhotosUI, UserNotifications, EventKit (Calendar)
- **Architecture**: MVVM, Protocol-Oriented
- **Design**: SF Symbols, カスタムグラデーション、ダークモード完全対応

## 📱 アプリの使い方

1. **サブスクの追加**
   - 「サブスク一覧」タブの右上にある「＋」ボタンをタップします。
   - 上部の「爆速クイック・人気サービスから追加」から該当するものを選択するか、「📸 スクショ・画像からAI自動入力」でカメラロールの画像を読み込ませると、フォームが自動で埋まります。
   - 必要に応じて金額や支払い周期（月額/年額）を微調整して保存します。
2. **コストの分析**
   - 「ダッシュボード」タブを開くと、現在の総額とカテゴリ別の割合が一目でわかります。
   - 右上のボタンで「月額換算」「年額換算」を切り替えることで、長期的なコストを把握できます。
3. **解約と一時停止**
   - 利用頻度が低いサブスクは、一覧から左スワイプして「削減」します。
   - 「削減履歴」タブに移動し、節約額のモチベーションに繋がります。再開したい場合は、右スワイプから「復元」をタップするだけで戻せます。

## 🚀 開発環境のセットアップ

1. リポジトリをクローンまたはダウンロードします。
2. Xcode 15以上で `SubsqManager.xcodeproj` を開きます。
3. Targetを `SubsqManager` に設定し、iOS 17対応のシミュレータまたは実機を選択します。
4. `Cmd + R` でビルド＆実行します。

> **Note**: アプリの機能を完全にテストするには、通知の許可およびカレンダーアクセスの許可が必要です。
